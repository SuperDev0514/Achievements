# 1) Quickdraw

### 🧠 Learn: **Create and close your issue**
### 🎁 Reward:
| Title | Badge | Earned by |
| --- | --- | --- |
`Quickdraw` | <img width="148" alt="quickdraw-default" src="https://user-images.githubusercontent.com/12344862/179227017-a2f0dfdd-6c3d-4371-acbb-b8dc71be4419.png"> |  Closed an issue or a pull request within 5 min of opening

### Learn how to create and close your issue and get an Quickdraw achievment!
1. Click [Create new Issue](https://github.com/Puliczek/github-achievments-academy/issues/new?assignees=&labels=Quickdraw&template=quickdraw---.md&title=%5BQuickdraw%5D+I+am+here+to+learn+how+to+open+and+close+issues+on+Github%21+%F0%9F%A4%A0)
(or go to **"Issues"** tab. Click **"New Issue"**. Get started with **"Quickdraw 🤠"** template)
2. Click **"Submit new issue"**
3. Wait 10 seconds
4. Click **"Close issue"**

**🎉 Congratulations - you just got an Quickdraw achievement! 😎**

# 3) Pull shark
### 🧠 Learn: How to create public repository, PR (Pull Request) and Merge PR (Pull Request)
### 🎁 Reward:
| Title | Badge | Earned by |
| --- | --- | --- |
`Pull shark` | <img width="148" alt="quickdraw-default" src="https://user-images.githubusercontent.com/12344862/182032489-5630ebbe-7441-4062-864d-efb8d9d8c942.png">|2 pull requests merged

### Learn how to how to create public repository, PR (Pull Request) and Merge PR (Pull Request) and get an `Pull shark` achievement!
1. Create new **PUBLIC** repository with README.md
![image](https://user-images.githubusercontent.com/12344862/183138853-560d0486-a27a-4da1-9121-cc13140a051e.png)

2. Edit README.md 

![image](https://user-images.githubusercontent.com/12344862/183139161-24747a33-33f7-4557-b385-9c3c7d8165cd.png)

3. Edit README.md and create a new PR

![image](https://user-images.githubusercontent.com/12344862/183139428-c2c50d65-6171-4556-89da-c9422b61a5f7.png)

4. Click `Create pull request`

5. Click `Merge pull request`
![image](https://user-images.githubusercontent.com/12344862/183140713-135824db-fe47-47f1-a0e6-6f41e2ab8c63.png)

6. Repeat 2-6 Step (you need to do it 2 times to get an achievement)

**🎉 Congratulations - you just got an Pull shark achievement! 😎**
